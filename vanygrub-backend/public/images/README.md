# VNY Toba Shoes - About Page Images Guide

Untuk menampilkan halaman About VNY Toba Shoes dengan sempurna, Anda perlu menambahkan gambar-gambar berikut ke dalam folder `public/images/`:

## Required Images:

### 1. History Section
- **`vny-toba-history.jpg`** (800x480px)
  - Gambar sejarah perjalanan VNY Toba Shoes
  - Bisa berupa foto founder, workshop pertama, atau milestone penting

### 2. Motif Batak Section
- **`motif-gorga.jpg`** (400x300px)
  - Foto detail motif Gorga pada sepatu atau kain tradisional
  
- **`motif-ulos.jpg`** (400x300px)
  - Foto detail motif Ulos pada sepatu atau kain tradisional
  
- **`motif-singa.jpg`** (400x300px)
  - Foto detail motif Singa pada sepatu atau kain tradisional

### 3. Philosophy Images
- **`craftsmanship-1.jpg`** (600x400px)
  - Foto proses keahlian tradisional pembuatan sepatu
  - Bisa berupa tangan craftsman sedang bekerja
  
- **`craftsmanship-2.jpg`** (600x400px)
  - Foto inovasi modern dalam pembuatan sepatu
  - Bisa berupa mesin modern atau teknologi terbaru

### 4. Vision & Mission Section
- **`vision-image.jpg`** (400x200px)
  - Gambar yang merepresentasikan visi perusahaan
  - Bisa berupa sepatu yang telah jadi dengan motif Batak
  
- **`mission-image.jpg`** (400x200px)
  - Gambar yang merepresentasikan misi perusahaan
  - Bisa berupa proses edukasi atau pelestarian budaya

## Alternative Options:

Jika Anda belum memiliki gambar-gambar tersebut, Anda dapat:

1. **Menggunakan placeholder images dari Unsplash:**
   - Shoes craftsmanship: https://unsplash.com/s/photos/shoe-making
   - Traditional patterns: https://unsplash.com/s/photos/traditional-pattern
   - Indonesian culture: https://unsplash.com/s/photos/indonesia-traditional

2. **Menggunakan stock images dengan tema:**
   - Sepatu tradisional Indonesia
   - Motif Batak/tradisional
   - Kerajinan tangan
   - Workshop pembuatan sepatu

3. **Sementara menggunakan placeholder warna:**
   - Background gradasi sesuai color scheme (amber, red, orange)

## Color Scheme:
- Primary: Amber (#f59e0b, #d97706)
- Secondary: Red (#dc2626, #b91c1c)
- Accent: Orange (#ea580c, #c2410c)
- Neutral: Gray (#374151, #6b7280)

## Notes:
- Pastikan semua gambar sudah dioptimasi untuk web (format WebP/JPEG)
- Resolusi minimal: 400x300px untuk detail shot, 800x480px untuk hero images
- Ukuran file maksimal: 200KB per gambar untuk performa optimal
